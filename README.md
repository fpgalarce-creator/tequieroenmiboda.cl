# tequieroenmiboda.cl

Plataforma web premium para vender invitaciones digitales de matrimonio personalizables.

## Stack
- Next.js App Router
- React + TypeScript
- Tailwind CSS

## Rutas principales
- `/`
- `/plantillas`
- `/plantillas/[slug]`
- `/plantillas/[slug]/personalizar`
- `/checkout`
- `/contacto`
- `/admin`

## Desarrollo
```bash
npm install
npm run dev
```

## Producción
```bash
npm run build
npm run start
```

> Checkout, panel admin y RSVP están implementados como mock funcional y preparados para integrar backend, almacenamiento y pasarela de pago.
